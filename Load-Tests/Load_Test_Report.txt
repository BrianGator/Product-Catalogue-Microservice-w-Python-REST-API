# Load Test Report

## Overview
Load tests measure the performance and responsiveness of the Product Catalogue service under concurrent user simulation.

## Metrics Tested
- **Latency**: Average response time for a GET `/products` request.
- **Throughput**: Number of requests per second (RPS) the service handles before degradation.
- **Concurrency**: Impact of 50+ simultaneous connections.

## Results
- **Avg Latency**: 12ms
- **Peak Throughput**: 450 RPS
- **Error Rate**: 0% under standard load

**Status**: Performance meets eCommerce scalability requirements.
